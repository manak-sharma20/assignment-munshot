# Amazon India Luggage Competitive Intelligence Dashboard

A project to scrape luggage brand data (Safari, Skybags, VIP, American Tourister) from Amazon India, process and extract themes/sentiment from reviews, and display insights in a Streamlit dashboard.

## Setup

1. Create a virtual environment: `python3 -m venv venv`
2. Activate it: `source venv/bin/activate`
3. Install dependencies: `pip install -r requirements.txt`
4. Install playwright browsers: `playwright install chromium`
